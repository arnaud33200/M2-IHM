Contacting the authors

For bug reports or comments on the library itself or about this document, please me an email at <yannick.jestin@enac.fr> . For comments and ideas about Ivy itself (protocol, applications, etc), please join and use the Ivy mailing list

If you report a bug, try to identify the causal path leading to the bug, and submit a trace of the problem, if possible, using the -DIVY_DEBUG property to produce a trace of the ivy execution.
